# Quick Challenge: map + filter Practice

## Objective
Practice:
- `.map()`
- `.filter()`
- Arrow functions
- Method chaining

No loops allowed.

---

## Starter Data

```js
const numbers = [4, 7, 10, 13, 18, 21, 24, 29];
```
Create a new file within unit-4 called `array-object-challenge.js`
---

## Part 1 — Filter Even Numbers

Create a new array:

```
evenNumbers
```

- Only include even numbers

---

## Part 2 — Double the Numbers

Create:

```
doubledNumbers
```

- Multiply every number by 2

---

## Part 3 — Square Only the Odd Numbers

Create:

```
squaredOdds
```

- First filter odd numbers
- Then square them
- Must use chaining

---

## Part 4 — Numbers Greater Than 15, Minus 5

Create:

```
adjustedNumbers
```

- Only include numbers greater than 15
- Subtract 5 from each remaining number
- Must use chaining

---

## Expected Output Example

```
Even Numbers: [4, 10, 18, 24]
Doubled Numbers: [...]
Squared Odds: [...]
Adjusted Numbers: [...]
```

---

## Rules

- Use only `.map()` and `.filter()`
- No `for` loops
- Do not modify the original array
- Use arrow functions
